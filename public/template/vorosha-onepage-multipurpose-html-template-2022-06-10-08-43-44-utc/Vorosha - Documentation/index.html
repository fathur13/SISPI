Installation

To install Vorosha on your web server, download the files from Themeforest and unzip them to the local disk on your computer. Inside the extracted folder you will see three folders named Vorosha - Documentation, Vorosha - Template and Flaticon. Vorosha - Template folder structure is as follows:

  • css - contains all verdor and main style css files
  • sass - contains all sass file.
  • fonts - contains icon fonts
  • img - contains all images used in the template
  • js - contains all vendor and custom js files
  • php - contains email sent php file
  • index.html - Home Page

Using an FTP Client (such as Filezilla or Total Commander), you will need to upload all of the contents of the Vorosha - Template folder to your server. And other files can be used according to your preferences.

HTML Structure

This template is based on a responsive and full screen layout. And it's built using HTML5 standards.

index.html

From beginning of the <body> tag till the ending </body> tag, all the codes and texts are used for the template structure. You can change any text from the template by navigating through the index.html file and localizing each line of text you want to change.

Like index.html, you can change any page included with this template starting from <body> tag till ending </body> tag.
